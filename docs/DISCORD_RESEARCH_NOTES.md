# Discord Research Notes

## What Strong Servers Do

Strong Discord servers do not win because they have the most channels. They win because a new member knows what to do in the first minute, the main channels feel active, and regular members have small reasons to return.

Research takeaways from current Discord docs and public server directory behavior:

- Use Community server features: rules, onboarding, Server Guide, moderation protections, and insights.
- Use Server Guide instead of treating every resource as a normal chat room.
- Keep default channels limited to the rooms every new tester needs.
- Make new members complete 3-5 simple tasks, not read a wall of instructions.
- Track real activity, not bumping, fake watch time, or reward farming.
- Make events repeatable: roll call, watch party, match test, clip review, setup clinic.
- Reward proof-based participation with capped claims and mod awards.

## Nitro / Boost Call

Do not pay for Nitro just to grow. Nitro helps the account look better and unlocks personal perks, but it does not create community activity.

Worth paying later:

```text
Nitro Basic: only if you want lightweight personal emoji/upload perks.
Nitro: only if you personally want bigger uploads, HD streaming, custom profiles, and the included Boosts.
Server Boosts: only after the Discord has real daily testers and the server brand is stable.
```

Best first paid move is not Nitro. It is a clean domain or landing redirect that always points to the Discord/app.
