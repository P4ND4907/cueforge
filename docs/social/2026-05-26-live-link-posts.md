## Reddit No-Link Comment

Use this first while the Reddit account is still in trust-recovery mode.

```text
For FPS audio problems, I would split the test into setup, tuning, and real match proof before changing EQ again.

Setup: confirm the actual Windows output, Discord output, mic input, and any Sonar/APO/Voicemeeter layers.
Tuning: make one safe starter change, not five at once.
Match proof: play one real round and write down whether footsteps, direction, comms, fatigue, or mic clarity changed.

That usually tells you whether the problem is the game mix, routing, mic gain, or the headset/IEM curve.
```
