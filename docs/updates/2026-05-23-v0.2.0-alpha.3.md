# CueForge v0.2.0-alpha.3

Codename: Seamless Engine Foundation

This update starts turning CueForge into one connected setup engine instead of a set of separate audio labs.

## What Changed

- Added CueForge State Core.
- Added Audio Chain Graph.
- Added Readiness Score v2.
- Added Profile Engine v2.
- Added Hearing Model v2 foundation.
- Added Auto Detect v2 handoff into the app brain.
- Added Conflict Detector for routing, enhancer, apply-target, and hearing-proof issues.
- Added One-Click Setup Flow entry from the Command Center.
- Added a native engine preview manifest for the future desktop processing engine.
- Added Release Pack v2 export for profile, chain graph, readiness, APO text, and apply path.

## Player Promise

One guided player setup, one profile brain, one readiness score, one export/apply path.

CueForge still does not silently change Windows drivers, routing, or APO configs. It builds the profile, shows the risks, and keeps native apply steps explicit.

## Test Notes

- Unit coverage now includes core chain graph, conflict detector, readiness score, profile engine, hearing model v2, and export schema.
- Browser smoke verified the Command Center, setup handoff, profile-brain apply action, and layout overflow.
