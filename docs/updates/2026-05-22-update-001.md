# CueForge Update 001 - Panda Lab Roadmap

Date: May 22, 2026.

## Short Version

CueForge has a real testing loop now:

```text
Open the app.
Run Setup Gate.
Run Self Test.
Post the setup chain.
Play one real match.
Export a packet or report.
Tell us what changed.
```

The Discord hub is live with onboarding, roles, starter tasks, and resource pages. The master roadmap is published in `docs/MASTER_PLAN.md`.
